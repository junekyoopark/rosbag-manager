Fixing the 3D view

"No transform path" means the 3D view's origin frame doesn't match the frames in your TF tree.

  1. Click the 3D space view panel inside Rerun to select it.
  2. In the Blueprint panel on the right, find Space origin.
  3. Type the root frame of your TF tree — usually map, odom, or world.
  4. Entities attached to that frame or any child frame will now appear.

Tip: if sensor data still doesn't appear, the bag is likely missing /tf_static which defines fixed sensor-to-body transforms.
